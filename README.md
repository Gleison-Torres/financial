🚧 Em desenvolvimento

## Tecnologias

- Python
- Django
- HTML5
- CSS3
- JavaScript
- PostgreSQL 
- Git / GitHub

## Funcionalidades

### Implementadas
- Interface inicial
- Página de Login
- Página de Cadastro
- Home do sistema
- Layout responsivo
- Estrutura base com Navbar e Footer

### Em desenvolvimento

- Cadastro de despesas
- Banco de dados

- Exclusão de despesas
- Dashboard financeiro
